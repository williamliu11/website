1.1.9 PLTW Flower Maker Game

We created an interactive flower maker for our 1.1.9 project. User is asked to input a style of flower, stem height, flower size, color, petal color, etc. The turtle draws the background, and draws a flower based on user input. Uses movement to move around and create, uses iteration to create multiple petals, and uses conditional execution with if statement multiple times.
